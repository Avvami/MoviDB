# MoviDB - The Movie Database App
`[Lost in development]`
This project is a comprehensive Android application that uses the TMDB API to enhance your movie experience. Create **custom personal lists**, **track what you've seen**, and easily search and filter for **what to watch next**.

## Tech Stack
- Jetpack Compose
- Room Database
- Retrofit + Moshi
- Dagger Hilt
- Coil
